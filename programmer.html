Game Programmer Job Description

Job Specific Requirements

  • Can program in Java Script and C Sharp
  • Have some tangible game programming experience (even if it is just a game jam or a hackathon we want to see something you have done)
  • Can use or quickly learn how to use Unity 3D
  • Can work well with others and communicate with non-coders on the team
it would be nice if you...
  • Are willing to relocate to where we are for the upcoming summer
  • Love video games as much as we do / are good at games so we can play together on breaks
  • Are already a master of Unity 3D

Some things you might find yourself doing as the game programmer include:

  • Figuring things out. For example:
    • We are working on the proof of concept and we need to find a way to import models into the game from 3Ds Max with one button click… how can we do this?
    • We are converting the proof of concept to a live beta and we need to build a chat system and an auction house… how are you going to go about doing this? Do we need to hire more people? How much should this cost? We will answer these questions but you should be ready to give us your input.
  • Identifying problems early on. We will have limited resources early on. As a result we need to know where to allocate our resources to make the highest impact. You need to help
